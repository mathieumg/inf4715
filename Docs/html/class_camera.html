Detailed Description

Camera interface.

Copyright (C) 2012 - All Rights Reserved All rights reserved. http://www.equals-forty-two.com

The camera class is responsible to view a scene from different angles. This camera class support the viewing and projection.

         up
         ^        dir (focus - position)
         |        /      
         |      /
         |    /
         |  /
         |/_________>strafe
         pos
Left-handed coordinate system
Remarks
the camera system is based on UVN system.
