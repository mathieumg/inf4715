Display an assertion (not meant to be used directly, use MACRO below)

Custom assert function that pops a windows that demand the user what to do. The windows print the expression being asserted and the user message. Then the user can break, ignore or exit the application. The assertion is also copied to the clipboard.

Parameters
[in]value: evaluation of the assertion, 0 or 1
[in]expression: string literal of the expression being asserted
[in]message: custom message explaining the assertion
[in]line: line of the assertion
[in]file: source file of the assertion
[in,out]ignored: flag indicating to ignore the assertion now or later
Returns
  • True => Assert
  • False => Do not assert
Remarks
Mostly useful when used with a macro (See GTL_ASSERT)
