Detailed Description

Atom base class interface.

Copyright (C) 2012 - All Rights Reserved All rights reserved. http://www.equals-forty-two.com The Vicuna Atom is the base element for any class that we want to use in the engine. It will add basic engine features to any class the derives from it.
