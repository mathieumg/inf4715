Detailed Description

World core interface.

Copyright (C) 2012 - All Rights Reserved All rights reserved. http://www.equals-forty-two.com The world core is responsible to store and keep track of all game world entities. It also facilitate the access to all world entities for lookup and processing. This core doesn't managed the lifetime of entities, but is responsible to registered them to corresponding systems.
