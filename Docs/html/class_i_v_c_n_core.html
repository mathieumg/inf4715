Detailed Description

Defines core components of the engine. Core components are initialized at startup and updated at every frame.

Copyright (C) 2012 - All Rights Reserved All rights reserved. http://www.equals-forty-two.com Simple interface used to register cores in the game runtime. NOTE: The templated VCNCore based class can fill this polymorphic behavior.
