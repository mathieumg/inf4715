const bool VCNRenderNode::UpdateWorldTransform ( )
overrideprotectedvirtual

Updates the world transformation based on parent transform. Return true if the derived class also needs to update transforms.

Updates the world transformation matrix. The matrix gets cached for faster usage afterward. Any changes to local transforms need to set to true for the computation to occur at next frame.

Returns
true if the world transformation was updated.
