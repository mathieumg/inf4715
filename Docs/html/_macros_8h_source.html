5 /// @brief Debug macros
6 ///
7 /// Common debugging macros. Many of these macros are similar to those provided
8 /// by MFC and are designed to allow for "MFC neutral" code (code than can be
9 /// compiled with or without MFC support). Other macros add even more debugging
10 /// facilities.
11 ///
12 /// Description of available macros:
13 ///
14 /// ASSERT : Identical to the MFC macro of the same name.
15 /// VERIFY : Identical to the MFC macro of the same name.
16 /// TRACE : Identical to the MFC macro of the same name.
17 /// INFO : Similar to TRACE but includes a preamble specifying the
18 /// file name and line number where the INFO is called as
19 /// well as forces a line break at the end.
20 /// DBG : Code contained within this macro is included only in none FINAL builds.
21 /// BREAK : Forces a break in _DEBUG builds via DebugBreak.
22 ///
23 /// Calling examples:
24 /// bool test = true;
25 /// TRACE( L"Testing [test]=%d\n", test );
26 /// VERIFY( test );
27 /// test = false;
28 /// VERIFY( test );
29 /// INFO( L"[test]=%d", test );
30 /// ASSERT( test );
31 /// ASSERT_MSG( test, "Asserted!" );
32 /// BREAK;
33 /// DBG( assert( 0 ) );
34 ///
