Samples\Modifiers\PointCache

The Point Cache modifier requires the FBX SDK to compile.

Version 2011.1 or newer is required.

The FBX SDK can be found at www.autodesk.com/fbx

Steps:

1. Download and install the FBX SDK version 2011.1 or newer
2. Add the FBX SDK include folder to the project settings
3. Add the FBX SDK lib folder to the project settings
4. Compile
